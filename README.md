# HADA Film

Kumpulan film lengkap dari berbagai jenis seperti hollywood, anime, drakor dan drama asia termasuk tv show.

## Tech Stack

- Laravel (Backend Framework)
- TMDB (RESTFULL API)
- Foundation Framework (CSS Framework)
- Blade (Frontend Framework)
